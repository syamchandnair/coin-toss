# Coin Toss Application

A simple, interactive web application to flip a virtual coin with engaging 3D animations.

## Features

- **3D Animation**: Realistic coin flip animation using CSS 3D transforms.
- **Random Outcome**: Fairly generates "Heads" or "Tails" using JavaScript's random function.
- **Visual Effects**: Features a glittering and zooming text animation to announce the result.
- **Custom Design**: Uses custom images for the Heads and Tails faces of the coin.

## File Structure

The application consists of the following files:

1.  **`index.html`**

    - The main entry point containing the HTML structure for the coin, buttons, and result display.

2.  **`style.css`**

    - Contains styles for the dark theme, 3D coin transformations, and keyframe animations for the result text.

3.  **`script.js`**

    - Handles the click events, calculates the random outcome, and manages the rotation logic to ensure the coin lands correctly.

4.  **Assets**
    - `heads.png`: Image used for the heads side.
    - `tails.png`: Image used for the tails side.

## Usage

Simply open `index.html` in any modern web browser and click the **Flip** button to toss the coin.
